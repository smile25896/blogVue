日文筆記：動詞變化

學習目標

學會日文動詞變化。

第一類動詞：

あ段：

否定形：飲のむ → 飲のまない　（不喝）
使役形：飲のむ → 飲のませる
　　　　例句：母ははは妹いもうとに紅茶こうちゃを飲のませる。　（媽媽讓妹妹喝紅茶）
被動形：飲のむ → 飲のまれる　（被喝）

い段：

連用形：飲のむ → 飲のみ
　　　　後面可接ます、たい、て、た等等，但て、た會有音變等等的狀況。

う段：

辭書形：飲のむ
　　　　也就是動詞的原形，又稱「終止形」。
連體形：飲のむ → 飲のむ+名詞
　　　　飲のむもの　（喝的東西）和飲のみもの（飲料）不同

え段：

假定形：飲のむ → 飲のめば　（如果喝的話）
命令形：飲のむ → 飲のめ　（喝！）
可能形：飲のむ → 飲のめる　（能喝。）

お段：

意向形：飲のむ → 飲のも　（喝吧！）

て形與た形

前面提到連用形提到，後面接て與た時，會有音變，在此講述變化規則。
く、ぐ　　：いて／いた、いで／いだ
す　　　　：して／した
う、つ、る：って／った
む、ぶ、ぬ：んで／んだ
行く　　　：行って（特殊變化）

第二類動詞：

否定形：食たべる → 食たべない　（不吃）
使役形：食たべる → 食たべさせる
　　　　例句：母ははは妹いもうとにケーキを食たべさせる。　（媽媽讓妹妹吃蛋糕）
被動形：食たべる → 食たべられる　（被吃）
連用形：食たべる → 食たべ
　　　　後面可接ます、たい、て、た等等，但て、た會有音變等等的狀況。
辭書形：食たべる
連體形：食たべる → 食たべる+名詞
　　　　食たべるもの　（吃的東西）和食たべもの（食物）不同
假定形：食たべる → 食たべれば　（如果吃的話）
命令形：食たべる → 食たべろ　（吃！）
可能形：食たべる → 食たべられる　（能吃。）
意向形：食たべる → 食たべよう　（吃吧！）

第三類動詞：

否定形：しない　　、来こない
使役形：させる　　、来こさせる
被動形：される　　、来こされる
連用形：し　　　　、来き
辭書形：する　　　、来くる
連體形：する＋名詞、来くる＋名詞
假定形：すれば　　、来くれば
命令形：しろ　　　、来こい
可能形：できる　　、来こられる
意向形：しよう　　、来こよう
